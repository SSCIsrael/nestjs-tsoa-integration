
## Description

[Nest](https://github.com/nestjs/nest) demo project (monorepo) with tsoa integration.

[Forked version](https://github.com/SSCIsrael/tsoa) of `tsoa` s being used as a dev dependancy. 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Swagger

```bash
tsoa swagger
```
Output file: `dist/swagger.json`
